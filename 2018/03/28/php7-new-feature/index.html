php7 new feature

Created at 2018-03-28 Updated at 2018-05-17

主要新特性

标量类型声明（string int float boolean），有强制和严格两个模式

返回值类型声明： function xxx(): array{ xxx; }

??操作符，简化?:三元操作符

太空船操作符<=>

$a 大于、等于、小于$b时返回： 1  0  -1

define定义常量数组

define('ABC', [1,2,3]);  echo ABC[1];

匿名类：使用new class

$app = new Application;
$app->setLogger(new class implements Logger {
    public function log(string $msg) {
        echo $msg;
    }
});

闭包：call（暂时没明白咋回事）

生成器： 使用return返回一个表达式； yield from xxx(xxx代表另一个生成器)

性能提升

  1. Zval由结构体变为联合体，降低内存占用（24字节->16字节）
  2. 数组结构优化，整块数组元素和hash表连在一起，被分配在同一块内存中，有效降低CPU Cache Miss
  3. 函数调用机制，优化了参数传递，减少了一些指令，提高执行效率
  4. Zend Engine优化
