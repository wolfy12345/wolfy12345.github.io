php7 new feature

主要新特性

标量类型声明（string int float boolean），有强制和严格两个模式

返回值类型声明： function xxx(): array{ xxx; }

??操作符，简化?:三元操作符

太空船操作符<=>

$a 大于、等于、小于$b时返回： 1  0  -1

define定义常量数组

define('ABC', [1,2,3]);  echo ABC[1];

匿名类：使用new class

$app = new Application;
$app->setLogger(new class implements Logger {
    public function log(string $msg) {
        echo $msg;
    }
});

闭包：call（暂时没明白咋回事）

生成器： 使用return返回一个表达式； yield from xxx(xxx代表另一个生成器)
